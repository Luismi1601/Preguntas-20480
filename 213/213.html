The website contains the following HTML markup. <div id="header" <ul class="menuItems"> <li>HOME</li> <li>PRODUCTS</li> <li>ABOUT US</li> </ul> </div> You need to ensure that the <li> elements are aligned horizontally. You write the following CSS markup in the stylesheet. .menuItems li { Target1: Target 2; } How should you complete the markup? Select and Place: Answer Area background- position block Target 1: display grid Target 2: inline list-style table Correct Answer: Answer Area background- position block Target 1: display grid Target 2: inline list-style table Section: Volume D Explanation/Reference: El elemento li representa un item de una lista. Esta puede ser ordenada (<ol></ol>) o no ordenada (<ul></ul>) El atributo background-position: Define en qué lugar del elemento seleccionado va a aparecer la imagen en cada elemento de lista. El atributo list-style: No es un atributo de li sino un atributo de ul que sirve para definir todas las propiedades de las listas El atributo display: Indica como se van a mostrar los elementos de la lista. En las opciones aparecen como posibles valores del atributo los siguientes: block : El elemento genera un cuadro de elemento de bloque. inline : El elemento genera uno o más cuadros de elemento en línea. grid : El elemento se comporta como un elemento de bloque y establece su contenido de acuerdo con el modelo de cuadrícula. table : Estos elementos se comportan como elementos HTML <table>. Define un cuadro de nivel de bloque. Como se nos indica que se quiere que los elementos de la lsita aparezcan en linea References: http://learn.shayhowe.com/html-css/creating-lists/
